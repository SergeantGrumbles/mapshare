\# MapShare nod



Allows sharing of map information between players in Zomboid multiplayer



\## Mod directory



The location depends on whether you are running with steam or not.



\### Steam



Create directory `c:\\Users\\<youusername>\\Zomboid\\Workshop\\MapShare\\Contents\\mods\\`



Clone repo into the above dir.... you should then end up with



`c:\\Users\\<youusername>\\Zomboid\\Workshop\\MapShare\\Contents\\mods\\mapshare\\media\\lua\\client\\`





\### No steam mode



Create directory `c:\\Users\\<youusername>\\Zomboid\\Workshop\\MapShare\\Contents\\mods\\`



Clone repo into the above dir.... you should then end up with



`c:\\Users\\<youusername>\\Zomboid\\mods\\mapshare\\media\\lua\\client\\`





\## No steam mode



Create a shortcut for ProjectZomboid64.exe



Add to target path debug and steam variables:



```

"C:\\Program Files (x86)\\Steam\\steamapps\\common\\ProjectZomboid\\ProjectZomboid64.exe" -nosteam -debug

```



This is the bat file to run as host.
